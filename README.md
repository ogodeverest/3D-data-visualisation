## Demo

 - [Live Demo](https://ogodeverest.github.io/3D-data-visualisation/)



## Run Locally

Clone the project

```bash
  git clone https://github.com/ogodeverest/3D-data-visualisation.git
```

Go to the project directory

```bash
  cd 3D-data-visualisation
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** Three.js
